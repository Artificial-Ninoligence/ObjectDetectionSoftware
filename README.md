 ## Introduction:

This project is inspired by Joseph Chet Redmon and his YOLOv5 Object Detection Framework that uses R-CNN as its backbone, "Multi-Class Classification" as its algorithm, and "Localization" as its feature. Object Detection Software is meant to pass the semester modules listed above at Code University of Applied Sciences in the spring semester 2021.

Object Detection Software is a software that simplifiesdetecting a a specific object given by the user. Thetechnologies used in this projectis as follows:

1. ### Django Web Framework
2. ### SQLite
3. ### Object Detection Framework (Model: YOLO Version 5)
4. ### Google Collab
5. ### VS Code
6. ### GitHub
